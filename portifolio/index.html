HTML

HTML (HyperText Markup Language) é a linguagem base da web. Ele define a estrutura de uma página, organizando elementos como títulos, parágrafos, imagens, links, listas, formulários e muito mais.

CSS

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas HTML. Com ele, é possível controlar cores, fontes, espaçamentos, layouts responsivos, animações e muito mais.
